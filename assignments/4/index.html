Calculator

A modern, interactive calculator built with HTML, CSS, and JavaScript, offering intuitive controls, keyboard shortcuts, and clean visual feedback.

Try Me!
Show / Hide Features
  • Core Calculator Functions: Supports chained operations with +, −, ×, ÷, decimals, clear, backspace, plus/minus toggle, percent, and equals, with basic error handling for invalid results.
  • Smart Expression Handling: Builds an expression string, tracks operators with a Set, replaces consecutive operators, prevents extra decimals per operand, and cleans up lone operator edge cases.
  • Keyboard Support: Maps keys (0–9, +, −, *, /, ., =/Enter, Backspace, Delete, Esc, %, P/p) to their respective buttons, triggering the same logic as mouse clicks and preventing default browser behavior.
  • Animated Button Feedback: Clicks trigger a glow-pulse underglow animation on buttons using a reusable .glow-pulse class and @keyframes for a polished, interactive feel.

Task Management App

Console-based Java Task Management System that allows users to create and manage tasks with a rich set of features powered by advanced data structures.

Show / Hide Features
  • Multi-User System: Register and switch users, with each user’s tasks stored separately using HashMap and HashSet.
  • Comprehensive Task Management: Create tasks with title, category, description, due date, priority, recurrence, and optional dependencies.
  • Advanced Data Structures: Includes linked lists (tasks, history, recurring), stack for undo/redo, BST for sorted tasks, heaps and priority queue for urgency, and a directed graph for task dependencies.
  • Dependency Handling: Add dependent tasks with automatic cycle detection (DFS) to prevent circular references.
  • Reporting: Export all tasks, recurring tasks, dependency graph, and priority queue into a user-specific .txt report.
