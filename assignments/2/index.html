Task Management App

https://github.com/jasant753/CSC220/tree/main/TaskManagerApp

Developed a Java-based Task Management System incorporating advanced data structures (e.g., graphs, trees, hash maps) to manage complex task operations like prioritization, dependencies, and analytics.

Features

User Management

  • Register and switch between unique users
  • User tasks stored separately using HashMap and HashSet

Task Creation

  • Each task has a title, category, description, due date, priority
  • Tasks can be marked as recurring
  • Tasks can be dependent on other tasks

Data Structure Integrations

  • DoublyLinkedList: stores tasks for each user
  • SinglyLinkedList: maintains a history log of task operations
  • CircularLinkedList: manages recurring tasks
  • Stack (ActionStack): supports undo/redo of task creation
  • BinarySearchTree: organizes tasks by due date for sorted traversal
  • MinHeap/MaxHeap: retrieves most/least urgent task
  • TaskPriorityQueue: builds a priority queue from tasks in the BST
  • TaskDependencyGraph: models task dependencies using a directed graph with cycle detection

Task Dependencies

  • When adding a task, user can specify if it depends on another task
  • Tasks are added to a directed graph
  • Cycles are detected using DFS and rejected to prevent circular dependencies

Reporting and Visualization

  • Menu option to export:
    • All user tasks
    • Recurring tasks
    • Task dependencies
    • Priority queue (sorted by urgency)
  • Output is written to a .txt file per user

Grand Archive Calculator

Created a simple C++ console-based hypergeometric probability calculator for use in the Grand Archive TCG. Grand Archive features various “spirits” that give you different options of drawing your initial hand. This app helps competitive players calculate the probability that you will open with an important card turn 1.

Features

Interactive Console Prompts

  • Users can set deck size and select draw mode
  • Guides the user with step-by-step prompts

Spirit (Draw) Modes

  • Regular (7 cards): Standard 7-card hand calculation
  • Serene/Fragmented (6 cards): Draw 6; option to bottom any number of cards and redraw that many (known as “Glimpse” in-game)
  • Fortuitous (7 cards): Glimpse 7; if you miss, bottom up to 7 blanks and redraw exactly that many. Optionally “level,” which forces a random discard of 1 card from your final 7

First vs. Second Turn Support

  • Displays probabilities for going first and second
