Task Management App

Collaborated on a Java-based Task Management System incorporating advanced data structures (e.g., graphs, trees, hash maps) to manage complex task operations like prioritization, dependencies, and analytics.

Github Link
Show / Hide Features

User Management

  • Register and switch between unique users
  • User tasks stored separately using HashMap and HashSet

Task Creation

  • Each task has a title, category, description, due date, priority
  • Tasks can be marked as recurring
  • Tasks can be dependent on other tasks

Data Structure Integrations

  • DoublyLinkedList: stores tasks for each user
  • SinglyLinkedList: maintains a history log of task operations
  • CircularLinkedList: manages recurring tasks
  • Stack (ActionStack): supports undo/redo of task creation
  • BinarySearchTree: organizes tasks by due date for sorted traversal
  • MinHeap/MaxHeap: retrieves most/least urgent task
  • TaskPriorityQueue: builds a priority queue from tasks in the BST
  • TaskDependencyGraph: models task dependencies using a directed graph with cycle detection

Task Dependencies

  • When adding a task, user can specify if it depends on another task
  • Tasks are added to a directed graph
  • Cycles are detected using DFS and rejected to prevent circular dependencies

Reporting and Visualization

  • Menu option to export:
    • All user tasks
    • Recurring tasks
    • Task dependencies
    • Priority queue (sorted by urgency)
  • Output is written to a .txt file per user
